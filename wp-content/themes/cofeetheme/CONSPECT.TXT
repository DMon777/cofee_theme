                    Мои заметки.
 Возникли трудности при создании постраничной навигации.Оказывается для того чтобы
 вывести постраничную навигацию для постов полученных с помощью new WP_Query()
 нужно получить текущую страницу :
 $paged = get_query_var( 'paged' ) ? absint( get_query_var( 'paged' ) ) : 1;

 И далле уже получить посты с помоью WP_Query
                         $posts = new WP_Query([
                                 'posts_per_page' => 2,
                                 'category_name' => 'post',
                                 'paged' => $paged
                         ]);
 И уже при выводе постраничной навигации в функцию paginate_links передаем пара-
 метром total именно наш полученный объект :
  'total'   => $posts->max_num_pages.

  Для вывода галлере создал новый тип поста с помощью функции register_post_type()
у этой функции много параметров. После ее объявления в меню админки появился новый
пункт меню - Галлерея, и теперь туда можно добавлять заголовки контент и миниатюры
к постам так же как и при добавлении нового поста в пункте меню - записи.
